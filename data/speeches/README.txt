CHANGELOG
---------

Compared to the version described in the LREC 2018 conference article, the following changes have been made:
- Better coverage, refined metadata, duplicates and interviews removed, minor typographic improvements
- Bundesregierung.xml: now includes selected speeches by Helmut Kohl
- Bundestagspräsidenten.xml: now includes speeches by Wolfgang Thierse
